## assignment 8 pygame

### prerequisites
 - python 3.13

### how to start
clone the repo
```sh
git clone https://github.com/amdRyzen5600X/gamedev_ass8.git
```

create a virtual environment
```sh
python -m venv venv
```

activate the virtual environment
```sh
source venv/bin/activate
```

download dependencies
```sh
pip install -r requirements.txt
```

run the game
```sh
python main.py
```
